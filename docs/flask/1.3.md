# 1.3 Hello World：你的第一行 Flask 代码
在项目文件夹下新建一个文件 app.py，填入以下内容：
```
from flask import Flask

# 1. 初始化：创建一个 Flask 实例
app = Flask(__name__)

# 2. 路由：告诉 Flask 哪个 URL 触发哪个函数
@app.route('/')
def hello_world():
    # 3. 响应：返回给浏览器显示的内容
    return '<h1>Hello, Flask!</h1><p>这是我的第一个 Web 页面。</p>'

if __name__ == '__main__':
    app.run()
```

代码拆解：
 * Flask(__name__)：这是 Flask 的核心。它让程序知道在哪里寻找模板和静态文件。
 * @app.route('/')：这是一个装饰器。它把根路径（/）和 hello_world 函数绑定在一起。当用户访问网站首页时，Flask 就会调用这个函数。
