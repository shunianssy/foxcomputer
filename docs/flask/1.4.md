# 1.4 运行与调试
虽然你可以直接运行 python app.py，但更推荐使用 Flask 自带的命令行工具。
1. 启动服务：
在终端输入：
flask run

你会看到一行提示：Running on http://127.0.0.1:5000。打开浏览器访问它，你就能看到你的页面了！
2. 开启调试模式（强烈建议）：
在开发阶段，如果你改了代码，肯定不想每次都手动重启服务器。开启**调试模式（Debug Mode）**后，代码一存盘，服务器会自动重启。
 * 临时开启：
 
# Windows
set FLASK_DEBUG=1
flask run

# macOS/Linux
export FLASK_DEBUG=1
flask run

注意： 永远不要在生产环境（正式上线时）开启 Debug 模式，因为报错信息会暴露你的代码细节，甚至允许远程执行代码。
