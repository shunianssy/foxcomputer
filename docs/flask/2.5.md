# 2.5 静态文件管理

图片、CSS、JavaScript 都属于静态文件。Flask 要求将它们放在项目根目录下的 static 文件夹中

引用方式： 永远不要手动写死路径 /static/style.css，而要使用 url_for

## 示例：
```html

<link rel="stylesheet" href="{{ url_for('static', filename='css/style.css') }}">
<img src="{{ url_for('static', filename='img/logo.png') }}">
```

## 到现在为止，你的项目目录应该是这样的：
```bash
my_flask_app/
├── venv/             # 虚拟环境
├── static/           # 静态资源 (css, js, images)
│   └── css/
├── templates/        # HTML 模板
│   ├── base.html
│   └── index.html
└── app.py            # 主程序
```
