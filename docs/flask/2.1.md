# 2.1 路由与变量规则（Routing）
路由是 Web 应用的指路牌。除了固定路径，我们经常需要处理动态内容，比如“用户个人主页”。
 * 动态路由： 使用 <变量名> 来捕捉 URL 中的一部分。
 * 类型转换： 你可以指定参数类型，防止非法输入。
| 转换器类型 | 说明 | 示例 |
|---|---|---|
| string | 默认类型，接受任何不带斜杠的文本 | /user/<name> |
| int | 接受正整数 | /post/<int:post_id> |
| float | 接受正浮点数 | /price/<float:val> |
| path | 类似 string，但可以包含斜杠 | /file/<path:subpath> |
代码示例：
@app.route('/user/<username>')
def show_user_profile(username):
    return f'用户: {username}'

@app.route('/post/<int:post_id>')
def show_post(post_id):
    return f'正在阅读文章 ID: {post_id}'
