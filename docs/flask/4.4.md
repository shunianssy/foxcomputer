4.4 用户认证：Flask-Login
做网站避不开“登录”功能。Flask-Login 帮你处理了最麻烦的部分：记住登录状态、保护需要登录后才能访问的页面。
from flask_login import LoginManager, login_user, login_required, current_user

login_manager = LoginManager(app)

@app.route('/dashboard')
@login_required  # 这是一个神奇的装饰器，没登录的人访问会直接被踢走
def dashboard():
    return f"欢迎，{current_user.username}"
