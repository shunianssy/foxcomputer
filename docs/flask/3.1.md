第三阶段：数据交互（让应用动起来）
3.1 Request 对象：获取用户的输入
当用户在浏览器里点击按钮或输入文字时，数据会通过 request 对象传给 Flask。你只需要导入并使用它。
 * 查询参数 (GET)： 获取 URL 问号后面的数据，如 ?page=1&search=flask。
   from flask import request

@app.route('/search')
def search():
    keyword = request.args.get('q', '默认搜索词') # 获取名为 q 的参数
    return f'你正在搜索：{keyword}'

 * 表单数据 (POST)： 获取 HTML 表单提交的数据。
   @app.route('/login', methods=['POST'])
def do_login():
    username = request.form.get('username')
    password = request.form.get('password')
    return f'欢迎回来，{username}'

 * 上传文件： 通过 request.files 获取上传的文件。
