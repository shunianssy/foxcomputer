# 1.2 环境准备：搭建你的实验室

为了不让不同项目的依赖包打架，我们必须使用虚拟环境（Virtual Environment）
1. 创建项目文件夹：
在终端（Terminal/PowerShell）执行：
mkdir my_flask_app
cd my_flask_app

2. 创建并激活虚拟环境：
 * Windows:
   python -m venv venv
venv\Scripts\activate

 * macOS / Linux:
   python3 -m venv venv
source venv/bin/activate

3. 安装 Flask：
pip install flask
