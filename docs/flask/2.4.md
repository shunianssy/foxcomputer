2.4 模板继承：拒绝重复劳动
如果你的网站有 10 个页面，每个页面都有相同的导航栏和页脚，你肯定不想写 10 遍。
1. 定义父模板 base.html：
<!DOCTYPE html>
<html>
<head><title>我的网站</title></head>
<body>
    <nav>导航栏</nav>
    <main>
        {% block content %}{% endblock %}
    </main>
</body>
</html>

2. 子模板继承：
{% extends "base.html" %}

{% block content %}
    <h2>这是首页特有的内容</h2>
{% endblock %}
