# 2.4 模板继承：拒绝重复劳动

如果你的网站有 10 个页面，每个页面都有相同的导航栏和页脚，你肯定不想写 10 遍。

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>{% block title %}我的网站{% endblock %}</title>
  <link rel="stylesheet" href="{{ url_for('static', filename='style.css') }}">
</head>
<body>
  <header>
    <nav>
      <a href="/">首页</a>
      <a href="/about">关于</a>
      <a href="/contact">联系</a>
    </nav>
  </header>

  <main>
    {% block content %}{% endblock %}
  </main>

  <footer>
    <p>&copy; 2024 我的网站</p>
  </footer>
</body>
</html>
```

### 子模板 `index.html`

```html
{% extends "base.html" %}

{% block title %}首页{% endblock %}

{% block content %}
  <h1>欢迎来到首页</h1>
  <p>这是首页的特有内容。</p>
{% endblock %}
```
