# 3.2 Response 对象：反馈与跳转

服务器处理完数据后，需要给用户一个结果

重定向 (Redirect)： 让用户跳转到另一个页面
```python
from flask import redirect, url_for

@app.route('/admin')
def admin():
    # 如果没登录，跳转到登录页
    return redirect(url_for('login')) 
```
注：url_for('login') 会自动根据函数名生成对应的 URL /login，这比写死路径更安全

返回 JSON： 编写 API 接口时的标配
```python
from flask import jsonify

@app.route('/api/data')
def get_data():
    return jsonify({"status": "success", "data": [1, 2, 3]})
```
