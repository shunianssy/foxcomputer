# 3.3 会话管理 (Session)：记住用户是谁

HTTP 协议本身是“无记忆”的（无状态）。为了让服务器知道“这个请求和上个请求是同一个人”，我们需要 Session

Flask 的 Session 是加密存储在浏览器 Cookie 中的

1. 设置密钥 (必须)：
```
app.secret_key = '你的秘密字符串' # 用于加密 Session
```

2. 使用 Session：
```
from flask import session

@app.route('/set_user/<name>')
def set_user(name):
    session['user'] = name  # 存入数据
    return '已记录身份'

@app.route('/get_user')
def get_user():
    user = session.get('user', '游客') # 读取数据
    return f'当前用户：{user}'
```
