# 5.3 RESTful API：前后端分离
如果你想给手机 App 提供数据，或者配合 Vue/React 使用，你需要返回 JSON 格式的 API
```python
@app.route('/api/v1/users/<int:id>', methods=['GET'])
def get_user_api(id):
    user = User.query.get_or_404(id)
    return {
        "id": user.id,
        "username": user.username,
        "email": user.email
    }
```
