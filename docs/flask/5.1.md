# 5.1 蓝图（Blueprints）：代码的“部门管理”

蓝图允许你将应用分解为不同的模块。例如，你可以把“用户中心”和“后台管理”分开存放

 * auth.py (处理登录注册)
 * main.py (处理主页和展示)
 * admin.py (处理后台逻辑)
 
#使用方法：

## 在 auth.py 中定义蓝图
```
from flask import Blueprint
auth_bp = Blueprint('auth', __name__)

@auth_bp.route('/login')
def login():
    return "登录页面"

# 在主应用 app.py 中注册蓝图
app.register_blueprint(auth_bp, url_prefix='/auth')

这样，你的登录地址就变成了 yourdomain.com/auth/login
```
